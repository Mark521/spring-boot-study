c+num+str === 107hello
c+str+num === ahello10
c+(num+str)== a10hello
(str+num)+c== hell10a

//编译失败 String初始化必须有""

##### d、逻辑运算符

| &    |          |
| ---- | -------- |
| &&   | 短路与   |

| ！   | 逻辑非   |

```java

```

## 一、Java语言的特点

### 1、面向对象

两个基本概念：类、对象

三大特性：封装，继承，多态

### 2、健壮性

吸收C语言的特点，但去掉了其影响程序健壮性的部分（如：指针、内存的申请与释放），提供一个安全的内存管理和访问机制

### 3、跨平台性

跨平台

### 4、核心机制

Java虚拟机、垃圾回收机制

## 二、Java语言环境信息

## 三、Java历史

1、1991年，Green项目，最初命名Oak

2、1996年，发布JDK1.0版本

3、1999年，Java分成 J2SE,J2EE,J2ME

4、2004年，发布里程碑版本JDK1.5，更名JDK5.0

5、2005年，变更名称J2SE JAVASE

6、2009年，被Oracle收购

7、2011年发布JDK7.0

8、2014年发布JDK8.0
